DIF-AccumulationTool

Differential Item Functioning (DIF) accumulation simulator for ordered polytomous tests

About this tool

This simulator shows how small Differential Item Functioning (DIF) effects at the item level can accumulate into systematic total-score differences between groups in graded response model (GRM) tests.
